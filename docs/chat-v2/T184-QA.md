### Якщо тести падають на `database disk image is malformed` з шляхом `database/database.sqlite`

Перевірте, чи не залишився **`php artisan config:cache`**: файл `backend/bootstrap/cache/config.php` (локально, не в git) може зафіксувати абсолютний шлях до SQLite і тоді `DB_DATABASE=:memory:` з `phpunit.xml` не застосовується. Видаліть `bootstrap/cache/config.php` або виконайте `php artisan config:clear`, після чого повторіть тести.

## Поведінка (коротко)

| Умова | Результат |
|--------|-----------|
| `ai_llm_enabled = false` | Не плануються відкладені відповіді бота через `RudaPandaRoomReplyScheduler`; VIP image job виходить одразу. |
| `rooms.ai_bot_enabled = false` | У цій кімнаті не поститься icebreaker; scheduler та VIP image для неї не виконуються. |
| Порожні поля model id у `chat_settings` | Ефективні id беруться з `config/services.php` / env; у відповіді адміну показано `ai_gemini_model_effective`. |
