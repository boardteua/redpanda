# T121 — локальний час повідомлень у стрічці (QA)

## Контракт

- **Кімната / WS:** `post_date` — Unix-секунди (UTC instant), уже в API та broadcast; `post_time` лишається в payload для сумісності, UI не покладається на нього для відображення.
- **Приват:** `sent_at` — те саме; `sent_time` — легасі.

## Реалізація

- **`formatChatMessageTime.js`:** `Intl.DateTimeFormat(undefined, …)` — локаль і TZ браузера (MDN).
- **Стрічка:** `ChatFeedMessageRow.vue` — час з `post_date`, `<time datetime>` = ISO UTC.
- **Приват:** `PrivateChatPanel.vue` — з `sent_at`.
- **Архів:** `ArchiveChat.vue` — дата+час з `post_date` (без примусового `uk-UA`).

## Автоперевірки

- `npm run test:format-chat-time` — PASS.
- `npm run build` — PASS.

## Ручна перевірка (timezone / locale)

1. Відкрити кімнату чату з відомим дописом; зафіксувати показаний час біля рядка.
2. Chrome DevTools → **⋯** → **More tools** → **Sensors** → змінити **Location** / **Timezone** (або змінити системний TZ) і перезавантажити сторінку — **той самий** `post_id` має показувати час, зміщений відповідно до нової TZ (миттєвість незмінна).
3. За потреби повторити для панелі привату та сторінки архіву.

Скріни за бажанням додати в `docs/chat-v2/artifacts/`.
