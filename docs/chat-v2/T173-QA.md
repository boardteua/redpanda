## Перевірки контракту

- **Room:** `GET /api/v1/rooms/{roomId}/messages?limit=N&before={cursor}`  
  - `meta.next_cursor`: `int|null` (мінімальний `post_id` у повернутому чанку)  
  - `meta.has_more_older`: `bool` (true якщо існує `post_id < next_cursor`)
- **Private:** `GET /api/v1/private/peers/{peer}/messages?limit=N&before={cursor}`  
  - `meta.next_cursor`: `int|null` (мінімальний `id` у повернутому чанку)  
  - `meta.has_more_older`: `bool` (true якщо існує `id < next_cursor`)
