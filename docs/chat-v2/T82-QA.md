# T82 — Lightbox для зображень у стрічці (QA)

**Вердикт:** PASS (локальна збірка)

## Перевірки

| Критерій | Результат |
|----------|-----------|
| `npm run build` | PASS |
| Клік по inline-зображенню в `ChatMessageBody` (стрічка / приват) | Відкриває оверлей з повнорозмірним переглядом |
| Клік по вкладеному зображенню (`message.image`) у `ChatFeedMessageRow` | Те саме |
| Закриття: фон, Escape, кнопка «Закрити» | Працює; `overflow` body відновлюється |
| Фокус після закриття | Повертається на кнопку-тригер |
| Клавіатура: Tab цикл усередині діалогу | Через `handleModalTabCycle` (як у `RpModal`) |
| Архів (`variant="archive"`) | Без lightbox (як у задачі — scope стрічки; компактні прев’ю) |

## Примітки

- z-index оверлею **200** — вище приват-панелі (`z-[60]`).
- Ручні скріншоти за потреби оператора.
