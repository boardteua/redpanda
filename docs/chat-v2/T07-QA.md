# T07 — QA (панель чату 320px)

## Функціонал

- Правий сайдбар **фіксованої ширини 320px** (`w-[320px]`), порядок вкладок як у [CHAT-PANEL-SIDEBAR.md](../board-te-ua/CHAT-PANEL-SIDEBAR.md): **Люди → Друзі → Приват → Кімнати → Ігнор**.
- **Порожні стани:** Приват — *«Немає нових повідомлень»*; Ігнор — *«Список ігнор порожній»*; Друзі → запити — *«Немає запитів у друзі»*; Друзі → активний — *«Список друзів порожній.»*
- **Кімнати:** список кнопок замість `<select>`; активна кімната підсвічена; зміна кімнати оновлює query `?room=` та стрічку (як раніше).
- **Мобільний (<768px):** панель **off-canvas** (з’їжджає вправо), затемнення фону, **Escape** закриває; після вибору кімнати панель закривається.
- **Десктоп (≥768px):** панель у ряду з чатом; кнопка з іконкою «люди» **перемикає** видимість колонки; **хрестик** також закриває.
- Іконки вкладок у зоні **≥44×44px** (`h-11 w-11`).
- **a11y:** у панелі `role="tab"` + `aria-controls` / `aria-labelledby` на `tabpanel`; стрілки **←/→**, **Home/End** у рядку вкладок; після відкриття off-canvas фокус на «Закрити панель», після закриття — повернення на кнопку відкриття; WS/мердж ігнорує повідомлення з **іншої** `post_roomid`, ніж поточна кімната.

## Автоматичні тести

```bash
cd backend && npm run build
cd backend && php artisan test
```

## Ручний сценарій

1. Десктоп: `/chat` — панель видима; перемкнути всі 5 вкладок; закрити хрестиком — чат на всю ширину; відкрити кнопкою з іконкою користувачів.
2. Вузький viewport (DevTools): панель за замовчуванням схована; відкрити кнопкою «люди»; клік по затемненню або Escape — закрити.
3. Вкладка **Кімнати** — перейти в іншу кімнату, переконатися що повідомлення й URL оновились.
