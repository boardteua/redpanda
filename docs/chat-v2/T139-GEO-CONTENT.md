# T139 — GEO / публічний контент: короткий контент-спек

**Задача:** T139 (вітальня `/`, узгодженість meta ↔ H1 ↔ lead ↔ JSON-LD FAQ ↔ `llms.txt`).

## Цільова аудиторія

- Дорослі користувачі, які шукають **живий український онлайн-чат** (гість або акаунт).
- Агенти пошуку та LLM, яким потрібен **стислий, несуперечливий** опис без вигаданих цифр.

## Primary keyword (за погодженням продукту)

- **«український онлайн-чат»** + бренд **«Чат Рудої Панди»** — природньо в title/description, без набивання.

## Voice / tone

- **Розважальний чат:** тепло, легкий гумор у межах ясності, **без** corporate stiffness.
- **Заборонено в індексованих полях:** нецензурна лексика, мова ворожнечі, «ми світовий лідер», вигадані статистики, юридичні гарантії, порівняння з конкурентами без джерел.

## Заборонені / обмежені формулювання

- Обіцянки цілодобової модерації або «гарантованої» безпеки.
- Вигадані цифри користувачів / повідомлень.
- Тексти, що суперечать OpenAPI або реальній поведінці API.

## Внутрішні настроєві ескізи (не для прод)

- Референс ритму української живої подачі — skill **MYTETS_MODE** (лише як джерело настрою; у prod-копі без пікантної лексики).
