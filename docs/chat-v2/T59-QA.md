# T59 — QA (швидкий пошук кімнат у сайдбарі)

**Статус:** PASS

## Реалізація

- `ChatRoomSidebar.vue`: поле `type="search"`, debounce **250 ms** на фільтр, клієнтський матч за `room_name` та `topic`, порожній стан «Немає кімнат за запитом», `aria-label` / `aria-describedby`, видимий `<label>`.
- API / OpenAPI не змінювались (повний список кімнат уже в SPA).

## Автоматична перевірка

- `npm run build` (Vite 7) — успішно.

## Ручний чекліст (клавіатура / UI)

1. Відкрити панель → вкладка «Кімнати» → **Tab** до поля пошуку.
2. Ввести частину назви або опису — список звужується без перезавантаження.
3. Очистити поле (нативне очищення або видалити текст) — знову повний список.
4. Запит без збігів — текст «Немає кімнат за запитом».
5. Вузький viewport (off-canvas): те саме поле доступне під вкладкою «Кімнати».

Скріншоти: за потреби оператора додати до PR або сюди.
