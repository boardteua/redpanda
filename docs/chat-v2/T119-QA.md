# T119 — мобільна кімната чату (QA)

## Зміни (каркас + композер)

- **`ChatRoom.vue`:** корінь **`h-full flex-1 min-h-0 overflow-hidden`** + **`md:flex-row`** — висота від батька (ланцюг нижче).
- **`spa.blade.php`:** viewport як у **legacy board.te.ua** — лише **`width=device-width, initial-scale=1`** (без `interactive-widget`: стабільніший **Chrome**).
- **`html` / `body.rp-body` / `#app` / `.rp-app`:** **`height: 100%`**, **`min-height: 100dvh`**, колонка **`flex`**, у **`#app` / `.rp-app`** — **`flex: 1 1 0`** (не **`1 1 auto`**) + **`min-height: 0`**, щоб **композер не обрізався** `overflow-hidden`.
- **`rp-chat-external-wrap`:** на `max-md` — **`min-h-0 overflow-hidden`**.
- **`ChatRoomMainColumn.vue`:** **`overflow-hidden`** на всіх breakpoints.
- **`app.css`:** мобільний композер — **`font-size: 1rem`** у полі вводу (уникнення автозуму iOS при фокусі); **`min-width: 0` / `overflow-x: hidden`** на рядку композера; **`overscroll-behavior-y: contain`** на стрічці (`ul[role='log']`); **`color-scheme`** на `html` за темою; кнопка **«Надіслати»** — **`appearance: none`**, **`background-color`**, токени **`--rp-chat-send-*`** узгоджені з **`--rp-primary`** і в світлій темі.
- **`ChatRoomComposer.vue`:** форма й рядок — **`min-w-0 w-full max-w-full`**.
- **`syncComposerInputHeight`:** лише висота textarea (не layout viewport).

## Чекліст ручної перевірки

1. **Десктоп і мобільний:** зона вводу (композер) **завжди видима** внизу колонки чату, не зникає за нижнім краєм вікна.
2. DevTools / девайс ≤767px: вертикальний скрол **лише** у списку повідомлень; хедер і композер **не** «плавають» посередині при скролі сторінки (сторінка не скролиться).
3. Порожнє поле вводу — візуально **одна лінія**; Shift+Enter — кілька рядків до ліміту, далі скрол усередині textarea.
4. Відкрити off-canvas сайдбар — після закриття лейаут кімнати без зламаних висот.
5. **iPhone Safari:** фокус у полі — **без стрибка масштабу**; кнопка надсилання **видима** і в **темній** темі; скрол історії **без підкручування** всієї сторінки.
6. **Chrome (мобільна емуляція / Android):** без стрибків висоти при скролі стрічки / клавіатурі.
7. **`npm run build`** — PASS (артефакт збірки).
