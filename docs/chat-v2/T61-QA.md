## Реалізація (коротко)

- **Кнопки панелі:** `v-if="!panelOpen"` у `ChatRoomHeader.vue` (mobile + desktop); при відкритій панелі з DOM прибрані, щоб не потрапляли в Tab.
- **Фокус:** у `beginOpeningPanel` та `selectRoom` після закриття панелі ціль фокусу — `#main-content`, якщо тригер був кнопкою панелі (після T61 ref кнопки зникає з DOM під час відкритого сайдбару).
- **Хедер:** без окремого фону/рамки картки; типографіка та кнопки на токенах `--rp-text-muted`, `--rp-chat-toolbar-bg` тощо поверх `--rp-chat-app-bg`.
