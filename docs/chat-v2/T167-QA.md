## Матриця приймання (бекенд)

- **Глобально off:** `PATCH` з `web_push_enabled: false` → `WebPushPreferenceEvaluator::shouldDeliverRoomWebPush` / приватний шлях не доставляє (юніт + feature).
- **Кімната muted:** рядок у `user_web_push_room_mutes` → room push не проходить фільтр.
- **Приват peer muted:** рядок у `user_web_push_private_peer_mutes` → private push не проходить фільтр.
- **Валідація:** неіснуючий `room_id` → **422**; `muted_private_peer_ids` містить власний `id` → **422**.
- **Гість:** `GET|PATCH /api/v1/push/notification-settings` → **403**.
