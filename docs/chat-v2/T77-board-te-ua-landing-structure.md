# T77 — аудит вхідної сторінки board.te.ua (гість)

Джерело: [https://www.board.te.ua/](https://www.board.te.ua/) у **ізольованому контексті браузера** (без cookies), знімок a11y-дерева через **Chrome DevTools MCP**, 2026-03-18.

## Скріншот

- `docs/chat-v2/screenshots/board-te-ua-landing-login-guest.png` — повна сторінка (гостьовий вигляд).

## Логотип

- URL на legacy: `https://www.board.te.ua/upload/orange.png`
- Локальна копія для референсу: `docs/chat-v2/screenshots/board-te-ua-orange-logo.png` (256×256 PNG).

## Будова (зверху вниз, спрощено)

1. **Шапка бренду** (лінк на головну):
   - зображення логотипу (`orange.png`);
   - **h1:** «Чат Рудої Панди»;
   - **h2:** «Український онлайн чат».
2. **Ліва колонка — форма входу:**
   - підпис «Ім`я користувача» + textbox;
   - «Пароль» + textbox;
   - текст «забули пароль?»;
   - дії «Увійти», «Реєстрація» (на скріні як текст/кнопки — уточнити семантику в DOM при імплементації);
   - кнопка «Увійти з facebook»;
   - «Зайти анонімно».
3. **Права колонка — контент:**
   - **h3:** заголовок новини (приклад на момент знімка: `org100h`);
   - абзац опису/новини;
   - посилання з іконками: Форум (`weekdays.te.ua`), Чат зізнання (`random.board.te.ua`), Архів (`?history=1`).
4. **Онлайн:**
   - **h3:** «Користувачі онлайн N» (на знімку **N = 6**).
5. **Сторонні блоки:** банери Google ads / CMP cookie — у redpanda за політикою продукту можна не відтворювати.

## Примітки для redpanda

- Лічильник онлайн на вітальні: **N** = користувачі **залогінені в чат** (presence **T20**), публічний API + оновлення з інтервалом.
- Динамічні тексти вітальні (заголовок новини, тіло) — з **T75** `landing_settings`, якщо вже заведено.
- Рекламу / CMP з legacy у **T77** не відтворюємо (поки без реклами).
