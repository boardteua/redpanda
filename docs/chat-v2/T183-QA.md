# T183 — QA: простій у кімнаті (icebreaker)

**Дата:** 2026-04-01  
**Вердикт:** PASS

## Що зроблено

- Додано job **`PostRudaPandaRoomIcebreakerJob`**: коли в кімнаті тиша ≥ N хв (за останнім **public** повідомленням від **не-бота**) — постить **одне** коротке питання від системного бота.
- Анти-флуд: per-room cooldown через `RateLimiter` (`ruda-panda:icebreaker:room:{id}`).
- Налаштування в `chat_settings`:
  - `ai_icebreaker_enabled` (bool)
  - `ai_icebreaker_idle_minutes` (default 60)
  - `ai_icebreaker_cooldown_minutes` (default 180)
  - `ai_icebreaker_jitter_minutes` (default 10) — додатковий випадковий поріг перед постом.
- Планувальник Laravel (bootstrap `withSchedule`): кожні 5 хв dispatch job на публічні кімнати (handler сам no-op, якщо вимкнено).

## Автотести

```bash
cd backend && php artisan test --filter RudaPandaRoomIcebreakerTest
```

Очікування: PASS (disabled → no-op; idle → пост; cooldown → не спамить).
