# T164 — QA (PWA іконки)

**Статус:** PASS (2026-03-29).

## Артефакти

- Каталог **`backend/public/pwa/`**: `icon-{48,96,192,512}.png` (**any**), `icon-{192,512}-maskable.png`, `apple-touch-icon-180.png`, `favicon-32.png`.
- Джерело растру: **`public/brand/board-te-ua-orange.png`**; фон заливки узгоджений з `background_color` маніфесту (`#f1f5f9`). Maskable: логотип ~62% сторони полотна по центру.
- Регенерація: **`cd backend && npm run pwa:icons`** (потрібен `sharp`).

## Маніфест і Blade

- **`vite.config.js`**: масив `manifest.icons` з окремими **`purpose: 'any'`** та **`'maskable'`**; `includeAssets` — усі файли з `pwa/` для прекешу SW.
- **`spa.blade.php`**: `link rel="icon"` PNG 32×32 + legacy `.ico`; при наявності збірки — `apple-touch-icon` 180×180.

## Команди

- `cd backend && npm run pwa:icons` — PASS  
- `cd backend && npm run build` — PASS  
- `cd backend && composer test` — PASS
