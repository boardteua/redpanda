# T41 — Сайдбар: вкладка «Люди» за замовчуванням

## Статус: PASS

## Автоматичні перевірки

- `npm run build` (Vite) — PASS після змін.

## Поведінка (код / ручний чекліст)

- Ключ `localStorage`: `redpanda-chat-sidebar-tab`. Якщо ключа немає або значення не з переліку `users` / `friends` / `private` / `rooms` / `ignore` — активна вкладка **Люди** (`users`).
- Після перемикання вкладки значення зберігається; після hard refresh відкривається остання збережена вкладка.
- Відкриття привату з сайдбару (`openPrivatePeer`) лишається на вкладці «Приват» і теж персиститься.

## Опційно для оператора

- Скрін після очищення ключа / у приватному вікні: видно активну «Люди» одразу після входу в `/chat`.
