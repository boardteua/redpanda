## Відхилення від legacy (свідомо)

- **Темна тема (`data-theme=dark` / system dark):** палітра не копіює orange.css (там немає окремого dark); лишаємо існуючі `--rp-*` для читабельності.
- **Пари нижче WCAG AA:** чистий `#f90` на `#361202` для довгих текстів — перевіряти в контексті; для коротких ніків/плашок прийнятно. Якщо розширити на багаторядковий текст — освітлити фон або потемнити fg.
- **Посилання в сайдбарі:** у legacy `a { color: white }`; у redpanda раніше був блакитний акцент — для паритету orange на світлій темі переведено на білі напівпрозорі тони (`--rp-chat-sidebar-link`).
