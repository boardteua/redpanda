# T45 — Хедер чату + сайдбар за замовчуванням на мобільному

## Статус: PASS

## Опис кімнати

- Поки **T44** не додає окреме поле `description`, у хедері показується **`topic`** з API (`RoomResource.topic`), як і раніше в `chatTopicLine` — до двох рядків, повний текст у нативному `title`.

## Автоматичні перевірки

- `npm run build` — PASS.
- `php artisan test` — 126 passed (без змін PHP).

## Поведінка

- У хедері чату **немає** «Вийти» та «Архів чату» (доступ лишаються з тулбару **ChatRoomComposer**).
- Хедер: токени **sidebar** (`--rp-chat-sidebar-*`), рамка `rounded-lg`, узгоджено з панеллю сайдбару.
- **Мобільний** старт: `panelOpen === true` після `initViewportListener` + явний `syncBodyScrollLock` при вузькому viewport — сайдбар (вкладка «Люди» з **T41**) видно одразу; закриття — X / overlay / Escape.

## Опційно для оператора

- Скрін: хедер без Вийти/Архів; `topic` у хедері; мобільний перший екран з відкритою панеллю.
