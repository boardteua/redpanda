**Вердикт:** PASS

## Доказ

- **PHPUnit:**
  - `cd backend && php artisan test --filter=RoomMemoryServiceTest`
  - `cd backend && php artisan test --filter=BotIdempotencyTest`

## Що перевірено

- Є персистентний стан **room-level** summary (`chat_ai_room_summaries`) з вказівником `summary_until_post_id`.
- `RoomMemoryService::rollupSummary()` робить **rolling summary** і зсуває вказівник, коли історія виходить за **time window** з налаштувань (`ai_summary_window_hours`, дефолт **3 години**).
- `RoomMemoryService::buildContext()` повертає `summary` + останні N повідомлень після вказівника.
- Є детермінований `client_message_id` від `IdempotencyKey::toClientMessageId()`; дубль для того ж `user_id` блокується **унікальним індексом** `chat_user_client_msg_unique`.
