# T27 — QA: сендвіч-меню (hamburger) у сайдбарі

## Зміни

- Меню дій користувача (T22) відкривається з **окремої кнопки «три смужки»** (`SidebarHamburgerTrigger.vue`): `aria-label`, `aria-expanded`, `aria-haspopup`, мінімальний розмір **44×44px** (`h-11 w-11`).
- Рядки «Люди» (я / онлайн), **Друзі**, **Приват** (поруч з аватаром), **Ігнор** — клік по імені/аватарі більше не відкриває меню; логіка пунктів і `UserBadgeContextMenu` без змін.
- Закриття по кліку поза: `UserBadgeContextMenu` ігнорує `mousedown` на `[data-rp-user-badge-menu-trigger]`; повторний клік по тому самому тригеру **згортає** меню.
