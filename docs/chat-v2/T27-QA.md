# T27 — QA: сендвіч-меню (hamburger) у сайдбарі

## Зміни

- Меню дій користувача (T22) відкривається з **кнопки «три смужки»** (`SidebarHamburgerTrigger.vue`): `aria-label`, `aria-expanded`, `aria-haspopup`, мінімальний розмір **44×44px** (`h-11 w-11`).
- Пункти показуються **інлайн під рядком** у жовтому блоці (`UserBadgeInlineActionPanel.vue`, токени `--rp-burger-accent-*`), за референсом board.te.ua — не плаваючий `position:fixed` дропдаун.
- Рядки «Люди» (я / онлайн), **Друзі**, **Ігнор** — клік по імені/аватарі не відкриває меню; **вкладка «Приват»** — без гамбургера (лише відкриття розмови з рядка).
- Матриця пунктів — спільний модуль `resources/js/lib/userBadgeMenuItems.js`.
- Закриття: клік поза панеллю та тригером; **Escape**; повторний клік по тому самому тригеру **згортає** панель.
