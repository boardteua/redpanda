# T31 — QA: модал «останні додані картинки»

**Вердикт:** PASS

## Доказ

- `cd backend && php artisan test` — усі тести зелені; зокрема `ChatImageApiTest` (список лише своїх зображень, пагінація та cap `per_page`, гість і неавторизований — 403/401).
- `cd backend && npm run build` — збірка без помилок.

## Реалізація (коротко)

- **GET** `/api/v1/images` — пагінація Laravel, `per_page` 1…60 (обрізання великих значень), сортування за `date_sent` та `id` спадно; **гість** — JSON 403.
- **UI:** кнопка в тулбарі композера відкриває модал з сіткою прев’ю; вибір встановлює той самий стан, що й після `POST /api/v1/images` (чернетка перед відправкою); порожній стан, Escape, клік по фону закриває; «Завантажити ще» для наступних сторінок.
- **OpenAPI:** `docs/chat-v2/openapi.yaml` v0.14.0 — операція `listMyChatImages`, схеми `ChatImageListItem`, `ChatImagesPaginatedResponse`.

## Ручна перевірка (опційно)

Зареєстрований користувач: кілька upload → відкрити «Мої зображення» → обрати картинку → прев’ю в композері → надіслати. Гість: кнопка та file-picker зображень вимкнені в UI.
