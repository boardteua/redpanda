# T16 — Візуальний паритет з board.te.ua (QA)

**Задача:** палітра чату, розміри, структура вікна (хедер, стрічка, композер, панель 320px) узгоджені з референсами в `docs/board-te-ua/screenshots/` та текстами `CHAT-MAIN-INPUT.md`, `CHAT-PANEL-SIDEBAR.md`, `SITE-STRUCTURE.md`.

## Автоматичні докази

```bash
cd backend && npm run build
cd backend && php artisan test
```

- Збірка Vite без помилок; PHPUnit — регресій немає.

## Ручний чекліст (браузер)

1. **`/chat`, desktop** — основна колонка на всю ширину (без `max-w-3xl`); стрічка займає доступну висоту в межах вікна (`md:h-[100dvh]`), прокрутка лише в зоні повідомлень.
2. **Палітра** — теплий фон стрічки (`--rp-chat-feed-bg`), окремі тони хедера, композера, сайдбару; темна тема — відповідні токени в `app.css`.
3. **Стрічка** — м’який «wash» по центру (аналог декоративного тла з `05-chat-viewport.png` без зовнішнього файлу маскота).
4. **Композер** — ряд інструментів над полем (B/I/U з `disabled`, фото, надсилання іконкою), placeholder узгоджений з `SITE-STRUCTURE.md` (⇧ / приват `/msg`).
5. **Сайдбар 320px** — фон і бордер через `--rp-chat-sidebar-bg` / `--rp-chat-chrome-border`.
6. **Контраст** — текст повідомлень і поля вводу залишаються читабельними (WCAG AA для ключових елементів; перевірити світлу/темну тему).

## Скріншоти (за потреби до релізу)

Підписати знімки посиланнями на файли референсу, напр.: «стрічка + композер — як `05-chat-viewport.png`», «сайдбар — як `sidebar-01-users.png`».
