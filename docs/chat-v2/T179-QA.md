**Вердикт:** PASS

## Доказ

- **PHPUnit:** `cd backend && php artisan test --filter=RudaPandaRoomReplySchedulerTest`

## Що перевірено

- Відповідь бота ставиться в чергу з **delay** (min/max у `chat_settings`, дефолт 1200–3000ms).
- Є **anti-flood** на кімнату і глобально через `RateLimiter` (ліміти з `chat_settings`).
- Текст приводиться до “натурального чату”: **1 абзац**, без переносів рядків, з лімітом довжини (`ai_bot_max_reply_chars`).
