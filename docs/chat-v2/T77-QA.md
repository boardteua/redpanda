# T77 — QA evidence (вітальня: візуальний паритет board.te.ua)

## Scope

- Оранжевий градієнтний фон сторінки та **одна біла картка** з тінню (токени `--rp-landing-*` у `resources/css/app.css`).
- **Шапка бренду** у картці: логотип `public/brand/board-te-ua-orange.png` (копія референсу), **h1** (заголовок з `landing.page_title` або дефолт), **h2** (підзаголовок / tagline).
- **Двоколонкова сітка** (lg): форма входу / реєстрації / гостя зліва; **новини + посилання** справа; плейсхолдери тексту, якщо адмін нічого не задав (узгоджено з T75).
- **«Забули пароль?»** — кнопка з поясненням (відновлення ще не реалізоване).
- Низ картки: **«Користувачі онлайн N»** як **h3**, **N** з `GET /api/v1/landing` (poll 45 с, як раніше).
- Тема (світла / темна / система) — кнопка на **оранжевому** тлі з контрастним стилем.

## Референс для порівняння скріном

- `docs/chat-v2/screenshots/board-te-ua-landing-login-guest.png`

```bash
cd backend && php artisan test --filter=LandingApiTest

cd backend && npm run build
```

## Ручний сценарій (оператор)

1. Відкрити `/` у браузері: видно оранжевий фон, білу картку, логотип і дві колонки на широкому екрані.
2. Порівняти з референсним PNG (сітка, наявність блоків форми / новин / онлайн).
3. За потреби: два сеанси з логіном у чат — переконатися, що **N** на вітальні оновлюється після poll (реал-тайм для T77 не обов’язковий).
