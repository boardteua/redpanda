# T124 — QA (композер: фокус після відправки)

## Автоматично

- `npm run build` у каталозі `backend/` — **PASS** (запуск під час закриття задачі).

## Ручна перевірка

1. Відкрити кімнату чату, сфокусувати поле вводу, надіслати **Enter** — після успіху курсор знову в тому ж `textarea` (DevTools: `document.activeElement.id === 'chat-composer'`).
2. Надіслати кнопкою «Надіслати» — те саме; клік по кнопці не залишає фокус на кнопці.
3. **Shift+Enter** — новий рядок; **Enter** без Shift — відправка; після відправки знову фокус у полі.
4. Приватна панель: після успішної відправки фокус у `textarea` привату (`#private-composer`).
5. Вузький viewport (мобільний): після send поле готове до вводу без зайвого тапу в поле.

## Примітка

Повернення фокусу виконується після `sending === false`, щоб не фокусувати `disabled` textarea.
