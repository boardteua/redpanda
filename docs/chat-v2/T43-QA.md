# T43 — Тема: з хедера в профіль / інфо для гостя

## Статус: PASS

## Рішення (гість)

- **Зареєстрований:** вкладка **Оформлення** у `UserProfileModal` — той самий цикл теми, що раніше в хедері (`localStorage` ключ `redpanda-theme`), **без** змін API / OpenAPI.
- **Гість:** повного профілю немає — блок **Оформлення** в `UserInfoModal` (режим «я»), також лише **локальний** `localStorage`.

## Автоматичні перевірки

- `npm run build` — PASS.
- `php artisan test` — PASS (126 tests), змін бекенду не було.

## Ручний чекліст

- У хедері чату **немає** кнопки перемикання теми.
- Профіль → **Оформлення** → кнопка змінює `data-theme` на `<html>` і зберігає вибір після F5.
- Гість: **Інформація** (про себе) → блок **Оформлення** з тією ж поведінкою.

## Опційно для оператора

- Скрін вкладки «Оформлення» в профілі.
