# T133 — QA (вітальня: тема + таби)

**Дата:** 2026-03-25

## Автоматичні перевірки

- `cd backend && npm run build` — **PASS** (Vite 7).

## Реалізація (коротко)

- Вітальня (`AuthWelcome.vue`): прибрано перемикач теми; таби «Вхід» / «Реєстрація» з `tabindex` за зразком roving tabindex, стрілки вліво/вправо/вгору/вниз, Home/End на `tablist`.
- Дефолт теми для сесії **без** `localStorage.redpanda-theme`: **`light`** через `getResolvedTheme()` у `chat/chatRoomConstants.js`, `App.vue`, `AuthCallback`, `ChatRoom` та інших екранах з читанням теми (Forgot/Legacy reset/Staff/Admin — лише узгодження початкового значення; перемикач теми на цих сторінках збережено).
- Збережений у `localStorage` вибір (`system` / `light` / `dark`) після входу в чат **не змінюється** цією задачею.

## Ручний smoke (оператор)

1. Нове вікно incognito → відкрити `/` → очікування: **світла** тема (`data-theme="light"` на `<html>`), **немає** кнопки теми на вітальні.
2. У чаті (профіль / модалка) перемкнути на темну → оновити сторінку → тема **зберігається**.
3. На вітальні: фокус на табах → стрілки перемикають вкладку й форму.

Скріншоти вузького/широкого viewport за бажанням додаються в PR; PII не включати.
