# T28 — QA: композер Enter / Shift+Enter

## Зміни

- У полі загального чату (`ChatRoom.vue`, `#chat-composer`): **Enter** (без Shift) викликає ту саму відправку, що й кнопка «Надіслати»; **Shift+Enter** залишає стандартну поведінку textarea (новий рядок).
- Під час **IME** (`isComposing` / keyCode 229) Enter не перехоплюється.
- Placeholder оновлено з підказкою про Enter / Shift+Enter.

## Ручний чекліст (desktop)

- У полі вводу: **Shift+Enter** додає перенесення рядка; текст лишається в полі.
- **Enter** без Shift: повідомлення відправляється (якщо є текст або вкладене зображення і кімната обрана), зайвий символ нового рядка не з’являється перед відправкою.
- Порожнє поле + Enter: без помилок, без зайвого запиту (логіка `sendMessage` як раніше).
