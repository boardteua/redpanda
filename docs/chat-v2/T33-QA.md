# T33 — QA: модал смайлів (пошук + сітка)

## Реалізація

- Новий компонент `ChatEmojiModal.vue`: Teleport, `role="dialog"`, підкладка (клік поза — закриття), **Escape**, циклічний **Tab** через `modalFocusTrap`, фокус у полі **«Шукати смайли»** після відкриття.
- Демо-каталог у `utils/chatEmojiCatalog.js`; вставка в композер у форматі **`:code:`** (legacy board.te.ua).
- Кнопка смайла в `ChatRoomComposer.vue` відкриває модал (disabled без обраної кімнати, узгоджено з полем вводу).
- Формат майбутніх асетів: [T33-EMOJI-ASSETS.md](./T33-EMOJI-ASSETS.md).

## Ручний чекліст

- Відкрити модал → ввести запит (напр. `vel`, `привіт`) — сітка фільтрується; порожній запит — усі демо-елементи.
- Обрати елемент — у полі з’являється `:code:`, фокус у textarea.
- **Escape** закриває модал; **Tab** не виходить за межі діалогу; після закриття фокус повертається на кнопку смайла.

## Автоматичні перевірки

- `npm run build` — PASS (під час закриття задачі)
- `php artisan test` — без змін API; повний прогін PASS
