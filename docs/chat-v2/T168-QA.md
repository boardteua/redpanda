# T168 — QA (мобільний chat-panel: бейдж приватних непрочитаних)

**Статус:** PASS (2026-03-29), автоматизовано; візуал — ручний smoke на вузькому viewport.

## Команди

- `cd backend && npm run build` — **PASS**

## Що зроблено

| Критерій | Результат |
|----------|-----------|
| Джерело лічильника | Той самий `totalPrivateUnread`, що й вкладка «Приват» / favicon (**T56**, **T65**) |
| Видимість | Бейдж лише на **мобільній** кнопці меню (`isNarrowViewport` + `md:hidden`), коли `!panelOpen` — збіг з **T61** |
| Вигляд | Класи як у `ChatSidebarTabBars` для таба private: червоний круг, cap `99+` |
| a11y | `aria-label` кнопки меню з урахуванням кількості непрочитаних (українською, як у таба) |
| Desktop | Друга кнопка панелі рендериться лише при `!isNarrowViewport` — без дубля бейджа на широкому екрані |

## Ручний smoke

1. Viewport ≤767px, сайдбар закритий: імітувати непрочитані привати → на іконці «гамбургер» з’являється бейдж.
2. Відкрити сайдбар / прочитати — лічильник зникає або зменшується узгоджено з табом «Приват».
