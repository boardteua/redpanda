# T127 — QA: модал смайлів біля кнопки (нижній лівий якір)

**Статус:** PASS (автоматичні перевірки нижче; скріншоти desktop / mobile — за потреби оператора в `docs/chat-v2/artifacts/`).

## Зміни

- `ChatEmojiModal.vue`: замість центрованого `RpModal` — портал із напівпрозорим фоном, панель `position: absolute` у повноекранному оверлеї; координати від `getBoundingClientRect()` кнопки смайлів; за нестачі місця зверху — відкриття **під** кнопкою; `max-height` обмежений, скрол усередині сітки; `resize` / `scroll` (capture) → перерахунок позиції.
- `ChatRoomComposer.vue`: `ref="emojiOpenBtn"` на кнопку, проп `:get-anchor="getEmojiAnchorEl"`.

## Перевірки (виконано)

- `cd backend && npm run build` — PASS.
- `cd backend && php artisan test` — PASS (342 tests).

## Ручний smoke (рекомендовано)

1. Кімната чату → кнопка смайлів → панель з’являється **зліва**, переважно **над** кнопкою (або під нею, якщо зверху мало місця).
2. Пошук, вибір смайлу → `:code:` у полі, фокус повертається в textarea (**T124**).
3. Escape, клік по фону — закриття; Tab циклить фокус усередині панелі.

## Примітка

Інтерфейс **LTR**; `RpModal` для інших модалок не змінювався.
