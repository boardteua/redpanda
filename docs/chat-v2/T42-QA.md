# T42 — Сайдбар: шапка без «Панель», таби в одному ряду з закриттям

## Статус: PASS

## Автоматичні перевірки

- `npm run build` — PASS.

## Ручний чекліст

- **Desktop:** у колонці сайдбару немає тексту «Панель»; іконки вкладок у верхній смузі; кнопка закриття справа в тій самій смузі.
- **a11y:** `role="tablist"` / `role="tab"` / `aria-selected` без змін; Tab-цикл по табах як раніше (`tabindex` 0 лише на активному); стрілки Home/End на `tablist`.
- **Mobile:** без змін (бургер-рядок як був).

## Опційно для оператора

- Скрін desktop + вузький viewport.
