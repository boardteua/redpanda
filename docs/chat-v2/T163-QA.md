**Статус:** PASS (2026-03-29).

## Команди

- `cd backend && npm run build` — **PASS** (генеруються `public/build/sw.js`, `public/build/workbox-*.js`, `public/build/manifest.webmanifest`).
- `cd backend && composer test` — **PASS** (після збірки Vite).

## Перевірки

| Критерій | Результат |
|----------|-----------|
| Web App Manifest (`name`, `short_name`, `start_url`, `scope`, `display`, кольори, іконки) | У `vite.config.js` → збірка `build/manifest.webmanifest` |
| Service Worker + прекеш shell (JS/CSS/woff2) | Workbox `generateSW`, `navigateFallback: null` — HTML з Laravel не в прекеші |
| Реєстрація SW у проді | `virtual:pwa-register` + `registerSW({ immediate: true })` у `app.js` лише при `import.meta.env.PROD` |
| Підказка «Встановити» | `RpPwaInstallPrompt.vue` (`beforeinstallprompt`, `role="region"`, кнопки) |
| Заголовок `Service-Worker-Allowed` для `/build/sw.js` | `public/.htaccess` + інструкція nginx у `T163-PWA-OPERATOR.md` |

## HTTPS / installable (ручна перевірка)

На **HTTPS**-оточенні (staging/production): DevTools → Application → Manifest + Service workers; за наявності критеріїв браузера — «Install» / «Add to Home Screen». Для повної відповідності **maskable** іконок — **T164**.
